# 使用DEMO

### $this->Obtain()的参数说明：


``` 
$obj->Obtain(市名称, 查询类型);
查询类型主要有以下五种：
0 ：当天天气
1 ：第二天
2 ：第三天
3 ：第四天
4 ：第五天
```
